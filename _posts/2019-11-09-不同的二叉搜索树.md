## 不同的二叉搜索树

![](../img/不同的二叉搜索树.png)

#### [96. 不同的二叉搜索树](https://leetcode-cn.com/problems/unique-binary-search-trees/)

#### 思路

动态规划

本次组成树的个数为每个数作为根节点数的个数之和。

```java
/*
G(n) = G(0)*G(n-1)+G(1)*(n-2)+...+G(n-1)*G(0)
*/
class Solution {
    public int numTrees(int n) {
        int[] nums = new int[n+1];
        nums[0] = 1;
        nums[1] = 1;
        for (int i = 2; i < n+1; i++) {
            for(int j = 1; j < i+1; j++){
                nums[i] = nums[i] + nums[j-1] * nums[i-j];
            }
        }
        return nums[n];
    }
}
```
